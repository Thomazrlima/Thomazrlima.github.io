SUBlime

Libreflix is an open-source video streaming platform designed to provide accessible and inclusive content. The subtitle project manages and provides translations in multiple languages for the hosted videos, aiming to make films accessible to a global audience. It utilizes Java, JavaScript, HTML, and CSS.

Pris.ME

Pris.ME is a white-label web platform aimed at assisting NGOs in efficiently managing their operations. Using Scrumban methodology in Jira, it ensures effective task management and progress tracking. It also features a custom reporting system, crucial for tracking and analyzing NGO performance metrics.
Kolekto

Kolekto is an online store specializing in collector items. Developed in Python with a frontend in HTML, CSS (Bootstrap), and JavaScript, and using Django, the app features SQLite for data storage and Azure for deployment. It offers a seamless shopping experience, user accounts, and secure transactions.
Mario Is Missing

An educational game inspired by the classic "Mario Is Missing" for Super Nintendo. The version here is implemented in C, using linked lists to create an interactive learning environment. Developed as part of an academic assignment and non-commercial, its main goal is to provide a fun platform for students to practice linked lists.

Archives.Sol

Archives.Sol is a comprehensive database repository that includes audio, photos, videos, and text. It is continuously updated with contributions from the community and is freely accessible to all interested users. The platform provides a valuable resource for research, archival, and creative projects.
Coffee-Break

Coffee Break is a project that helps manage coffee levels in containers like thermal carafes. A scale connected to an Arduino measures the weight of the container and coffee, alerting users when the coffee is low and displaying this information on a screen.
